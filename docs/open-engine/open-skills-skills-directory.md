# Open Skills directory.

The Open Skills directory organizes reusable agent primitives by the kind of work they make repeatable. Each category opens into the skills themselves: what they do, what they depend on, and how they install.

These are not downloadable finished skills. Each entry gives you the prompt to build the primitive with your own agent, defaults, stack, and taste, so the result belongs to your workflow instead of ours.

Choose the category first. Then install the smallest primitive that makes the next workflow repeatable.
